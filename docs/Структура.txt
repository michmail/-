main.py
├── input_output.py (ввод данных, вывод результата)
├── calculator.py (расчет стоимости цветов)
├── discounts.py (скидки и доп. услуги)
└── logger.py (логирование расчетов)


## Взаимодействие модулей

1. main.py вызывает input_output.get_user_data() → получает словарь с данными
2. main.py передает данные в calculator.calculate_base_cost()
3. main.py передает данные в discounts.calculate_extras()
4. main.py передает данные в discounts.apply_discount()
5. main.py передает итог в input_output.show_result()
6. main.py передает данные и итог в logger.log_calculation()
